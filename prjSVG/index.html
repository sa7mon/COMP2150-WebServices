SVG Information

  • What is SVG?
    • SVG is a Standard Vector Graphic. It's an image file that can easily be stretched without looking bad like most pictures do. It can do animations as well.
  • What are three benefits of SVG?
    • Since an SVG file is just a bunch of coordinates, it's very bandwidth friendly. The load times are the same if the image is small or huge.
    • They are incredibly easy to edit. There is no need for Photoshop knowledge to change the text in a picture - just open it up in a text editor.
    • Since it is code at it's core, one image can be turned into hundreds of different images by using the same core code and tweaking it.
  • What is one disadvantage of SVG?
    • It can be incredibly complicated to create intricate images. SVG is good if your image has lots of simple shapes in it, but there's virtually no way that you could re-create a hi-def picture of a bee on a plant with an SVG image.
