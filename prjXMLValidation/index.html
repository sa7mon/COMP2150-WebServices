Executive Summary

    The XML language is an all-purpose langauge made to store information. It was designed to be easily read by a parser, making the information easy to extract. It makes it very easy to pass and store information that will be used by many different platforms. Also built into XML is the ability to "validate" an XML document to make sure it's following certain rules in regards to what can be stored and how.

    The XML parser is what does all the heavy-lifting in reading XML documents. It converts XML data and elements into whatever data types you need for the platform you're on. A good example is the XML parser built into Android. Behind the scenes in every Android app is an XML doucment that describes the physical layout of the app. Google has made a set of rules that this XML document must abide by.
